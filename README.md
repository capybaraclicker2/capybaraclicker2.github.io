## 🛠️ Local Development & Deployment

This portal is built using lightweight, static assets (**HTML5, Vanilla CSS, and JavaScript / jQuery**).

### Running Locally
To run the project locally, start a simple HTTP server in the root directory:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server .
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Hosting on GitHub Pages
This portal is configured to work out-of-the-box with **GitHub Pages**:
1. Push this folder to your repository (e.g., `capybaraclicker2/capybaraclicker2.github.io`).
2. Go to Repository Settings -> **Pages**.
3. Set the source branch to `main` (or `master`) and folder to `/ (root)`.
4. Save, and your portal will be live at `https://capybaraclicker2.github.io/`.
